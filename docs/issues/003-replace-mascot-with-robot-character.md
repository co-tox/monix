# Issue: 마스코트 교체 — 고양이 ASCII → 로봇 픽셀 캐릭터

## 요약

`render.py`의 `_MASCOT` 고양이 ASCII 아트를 아래의 로봇/사이클롭스 픽셀 캐릭터로 교체합니다.
서버 모니터링 에이전트의 정체성에 맞게 기계적이고 기술적인 느낌을 주도록 변경합니다.

## 현재 상태

`monix/render.py:11`의 `_MASCOT` 리스트:

```python
_MASCOT = [
    r"  /\_/\  ",
    r" ( o . o)",
    r"  > Gemini <",
    r"  ~(___)-",
]
```

## 변경 목표

아래 로봇 캐릭터로 교체합니다. 외눈(`[O]`) 사이클롭스 로봇으로 서버 감시자 이미지를 표현합니다.

```
     ▄▄▄▄▄
   ▄███████▄
  ███████████
  ████[O]████
  ███████████
  ███████████
  █▀▀█▀▀█▀▀█
  █  █  █  █
  ███████████
   ▀███████▀
```

- 터미널 폭 변화에도 레이아웃이 깨지지 않도록 문자 너비를 일정하게 유지합니다.
- `render_welcome()`의 박스 레이아웃 내 `_text()` 렌더링을 통과하므로 ANSI escape 없이 순수 텍스트로 구성합니다.
- 기존 `_MASCOT` 라인 수(4줄)보다 늘어나는 경우 `render_welcome()`의 박스 줄 수에 영향이 없는지 확인합니다.

## 수정 대상 파일

- `monix/render.py` — `_MASCOT` 리스트 교체

## 완료 기준

- `monix` 실행 시 환영 화면에 로봇 캐릭터가 정상 출력됨
- 터미널 너비 80·100·110 기준으로 박스가 깨지지 않음
- 기존 테스트 전체 통과
